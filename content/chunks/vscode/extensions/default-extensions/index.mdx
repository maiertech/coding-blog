🍪 User extensions in [@code](https://twitter.com/code) are default extensions.
But they do not sync with containers. Use

`github.codespaces.defaultExtensions`

in `settings.json` for [@GitHub](https://twitter.com/github) Codespaces and

`remote.containers.defaultExtensions`

for containers to define and install default extensions.
