## Cross posting

I am not [using dev.to as a CMS](https://www.swyx.io/devto-cms/). The single
source of truth are my MDX files in the GitHub repo.

For the time being, I cross-post manually with this workflow:

1. Do a final edit on the MDX file and publish it.
1. Run `npx prettier --prose-wrap never --write <path/to/file>` to eliminate
   line breaks.
1. Copy the MDX code over to dev.to.
1. Remove frontmatter date and frontmatter tweets.
1. Add `published: false`.
1. Add `canonical_url`.
1. Save and double check.
1. Set `published: true`.
1. Pin post.
1. Submit tweet suggestion.
