## Shortcuts

Use ⌃ ⇧ / to open the shortcut explorer.

### Essential

| Shortcut         | Description                         |
| :--------------- | :---------------------------------- |
| ⌘ \              | Show/hide UI. A quick way to focus. |
| ⌥ 1              | Show layers panel.                  |
| ⌥ 2              | Show assets panel.                  |
| ⌘ G              | Group selection.                    |
| ⌘ R              | Rename.                             |
| ⌥ click and drag | Duplicate something.                |
| ⌥ move mouse     | Display distances.                  |
| ⌘ click          | Smart select the element you want.  |
| ^ C              | Pick color.                         |
| ^ G              | Toggle layout grid visibility.      |

### Placement

| Shortcut | Description          |
| :------- | :------------------- |
| ⌥ ↑      | Move up 1 pixel.     |
| ⌥ ⇧ ↑    | Move up 10 pixels.   |
| ⌥ ↓      | Move down 1 pixel.   |
| ⌥ ⇧ ↓    | Move down 10 pixels. |

### Tools

| Shortcut | Description                      |
| :------- | :------------------------------- |
| F        | Frame tool.                      |
| O        | Ellipse tool. Hold ⇧ for circle. |
| P        | Pen tool.                        |
| T        | Text tool.                       |

### Components

| Shortcut | Description                           |
| :------- | :------------------------------------ |
| ⌥ click  | Duplicate from main component.        |
| ⌥ ⌘ B    | Detach component from main component. |

### Text

| Shortcut | Description       |
| :------- | :---------------- |
| ⌘ B      | Toggle bold.      |
| ⌘ I      | Toggle italic.    |
| ⌘ U      | Toggle underline. |
