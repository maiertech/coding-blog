## CSS Modules

This is Next.js's default. CSS Modules help generate scoped CSS by generating
globally unique class names from local class names. The convention is
`[name].module.css` to trigger processing with CSS Modules.

## PostCSS

Next.js processes CSS with PostCSS, which can be customized with a
`postcss.config.json`.

## Global Styles

Import global styles in `pages/_app.js`.
